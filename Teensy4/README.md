# Usage

To use this project, you must have:

1. A Raspberry pi 4 with 8GB of memory
2. A Teensy 4.0
3. A 4 channel ESC
4. A BNO055 IMU
5. A TFMini Plus Lidar (or equivalent)
6. 4s brushless motors
7. a frame of your choice (files included)
